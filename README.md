# bingo

A React (Vite) Bingo game.

- Pick an odd grid size: 3×3, 5×5, 7×7, 9×9, or 11×11.
- The grid auto-fills with unique random numbers (1…N²).
- Click cells to mark them.
- Completing a full row, column, or diagonal draws a glowing line and turns those cells green.
- Complete **N lines on an N×N grid** (e.g. 5 lines for 5×5) to win — a **BINGO!** banner appears.

## Run locally

```bash
npm install
npm run dev
```

Then open the printed localhost URL (default http://localhost:5166).
